# 学习`LWJGL `

### 1. 创建`GLFW`窗口

创建`GLFW`窗口类, 定义必要的成员变量. 通过`InstanceHolder`的方式创建单例.

窗口类包含两个主要的方法:

| 方法       | 描述   | 内容                                       |
| -------- | ---- | ---------------------------------------- |
| `init()` | 初始化  | 包括初始化`GLFW`, 配置`GLFW`, 以及创建窗口.<br>然后使用`OpenGL`渲染当前窗口. |
| `loop()` | 主循环  | 主循环会周而复始的重复执行, 一般会在主循环里轮询事件`glfwPollEvents()`, 并且交换缓冲区`glfwSwapBuffers(long window)`渲染画面. |
